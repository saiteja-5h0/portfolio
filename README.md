# Sai Teja Portfolio

A custom dark editorial developer portfolio built with HTML, CSS, and JavaScript.
It has no database or server dependency and can be deployed directly to Vercel.

## Update your details

- Main text, email, navigation, and social links: `index.html`
- Projects, internships, education, certifications, and skills: `script.js`
- Colors and visual styling: `style.css`

Social and résumé links are already connected. Project buttons currently open the
GitHub profile; replace those three URLs with the individual repository or live-demo
links when available.

## Run locally

Open `index.html` directly, or serve the folder with any static file server.

## Deploy to Vercel

Import this folder as a new Vercel project. Keep the framework preset as **Other** and
leave the build command empty.
